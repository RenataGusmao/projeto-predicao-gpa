# EduPredict AI - Frontend

Site estatico para demonstracao de uma aplicacao de Data Science voltada a predicao de desempenho academico.

## Sobre o projeto

O sistema simula a estimativa do GPA final de um estudante a partir de informacoes como:

- GPA anterior
- horas de estudo tradicional
- uso semanal de IA generativa
- habilidade em prompts
- dependencia percebida de IA
- ansiedade em provas
- retencao de habilidades
- risco de burnout

Nesta versao, a predicao e calculada diretamente no navegador com JavaScript, sem necessidade de backend ou API externa. Isso permite publicar o projeto como site estatico no Vercel.

## Tecnologias

- HTML
- CSS
- JavaScript

## Como publicar no Vercel

1. Suba esta pasta `frontend` para um repositorio no GitHub.
2. No Vercel, clique em `Add New Project`.
3. Importe o repositorio do frontend.
4. Mantenha as configuracoes padrao de projeto estatico.
5. Clique em `Deploy`.

## Execucao local

Basta abrir o arquivo `index.html` no navegador.
